## Component A — Scientific Content (50 marks)

### A1: Research Question & Objectives (10)

| Mark | What you're seeing |
|---|---|
| **9–10** | Precise, compelling research question; significance is convincing; specific aim is unambiguous |
| **7–8** | Clear question, well-motivated; minor imprecision OK |
| **6** | Question present but generic or imprecise; aim not fully scoped |
| **4–5** | Question implied but not articulated; aim confused with methods/results |
| **0–5** | No identifiable research question; purpose of study unclear |

### A2: Methods (10)

| Mark | What you're seeing |
|---|---|
| **9–10** | All tools/DBs named with versions; key parameters stated; logical pipeline; reproducible |
| **7–8** | Mostly complete; minor omissions (a version, a date); broadly reproducible |
| **6** | Major tools named but details missing; general approach clear but not reproducible |
| **4–5** | Superficial; key tools/DBs missing; reads as narrative not a protocol |
| **0–5** | Absent or incoherent; cannot determine what was done |

### A3: Results (15)

| Mark | What you're seeing |
|---|---|
| **13–15** | Led by findings (not methods); all figs/tables cited and self-contained; precise, quantified; logical build |
| **11–12** | Key findings clear; figs well-integrated; minor precision gaps |
| **8–10** | Findings understandable but vague language; figs may have gaps; discussion bleeds in |
| **6–7** | Incomplete/disorganised; vague; figs missing or not cited; results/discussion blurred |
| **0–5** | Absent or incoherent; raw outputs with no narrative |

### A4: Discussion & Conclusions (15)

| Mark | What you're seeing |
|---|---|
| **13–15** | Confident interpretation; literature actively engaged (not just cited); specific limitations; conclusions directly answer research question |
| **11–12** | Accurate interpretation; literature cited with meaning; limitation noted; clear conclusion |
| **8–10** | Somewhat descriptive; literature cited but not explained; generic limitation or none; conclusion present but loosely tied to aims |
| **6–7** | Largely restates results; minimal literature engagement; no limitation; vague or unsupported conclusion |
| **0–5** | Absent or indistinguishable from results; no interpretation |

## Component B — Structure & Format (15 marks)

| Mark | What you're seeing |
|---|---|
| **13–15** | All sections present, ordered, proportioned; abstract covers all 4 elements and is within limit; word count within range; title precise; keywords appropriate |
| **11–12** | All sections present; abstract mostly effective; word count within or close to range; minor proportionality issues |
| **8–10** | All major sections present; one poorly proportioned or too brief; abstract has gaps; word count outside range |
| **6–7** | Section(s) missing or near-absent; substantially deviates from format; word count significantly out of range |
| **0–5** | Does not follow prescribed format; sections absent/scrambled |

**Quick checks for Component B:**
- [ ] Title — is it informative (what/how/found)?
- [ ] Abstract — does it cover: context / approach / key finding / significance? Under 200 words?
- [ ] Keywords — 4–6, relevant, searchable?
- [ ] All sections present: Intro / Methods / Results / Discussion / References?
- [ ] Word count: 2,000–2,500? (Check PDF word count tool or ask student to confirm)
- [ ] Supplementary: Are all Supp items cited in the main text? Do they have legends?

## Component C — Written Communication (20 marks)

### C1: Clarity & Precision (10)

| Mark | What you're seeing |
|---|---|
| **9–10** | Consistently clear, precise, well-structured sentences; appropriate for audience; confident; information-dense without being obscure |
| **7–8** | Mostly clear; occasional ambiguity; good control of technical language |
| **6** | Generally understandable; inconsistent clarity; some vague or imprecise passages |
| **4–5** | Significant clarity issues; reader must work to extract meaning; technical terms imprecise |
| **0–5** | Largely incomprehensible |

### C2: Grammar, Spelling & Register (10)

| Mark | What you're seeing |
|---|---|
| **9–10** | Essentially error-free; academic register throughout; consistent tense; abbreviations defined |
| **7–8** | Occasional errors; register mostly maintained; professional overall |
| **6** | Errors present but do not impede understanding; some informal phrasing *(apply language equity — see note below)* |
| **4–5** | Frequent errors that distract; register often inappropriate |
| **0–5** | Pervasive errors that prevent comprehension |

> **Language equity note (C2):** Do not penalise grammatical patterns that are consistent with second-language English writing if the meaning is clear. Apply C2 reductions only where errors impede comprehension or where register is fundamentally inappropriate (e.g., colloquial phrasing throughout, first-person informal narration in Methods).

## Component D — Literature Engagement (15 marks)

| Mark | What you're seeing |
|---|---|
| **13–15** | Predominantly primary peer-reviewed literature; directly relevant; Harvard correct; Discussion explains what papers found and how it relates — not just a citation; broad and selective |
| **11–12** | Mostly primary literature; relevant; minor formatting errors; literature engaged meaningfully in Discussion |
| **8–10** | Relevant but leans on reviews/textbooks; formatting inconsistent; cited in Discussion but papers not explained |
| **6–7** | Limited, low-quality, or irrelevant references; heavy use of websites; sparse/inappropriate citations |
| **0–5** | References largely absent, or non-peer-reviewed throughout |
